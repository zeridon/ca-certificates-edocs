# ca-certificates-edocs
[eDocs Bulgaria](https://sign.edocs.bg/documents/) root certificates

This package provides [eDocs Bulgaria](https://sign.edocs.bg/documents/) (sub-)root certificates in PEM format. Root certificates allow SSL-based applications to check for the authenticity of certificates issued by the eDocs Bulgaria authority.

[eDocs Bulgaria](https://sign.edocs.bg/) is a Bulgarian official eID issuer with full eIDAS compliance.

Precompiled debian style packages can be found on the [Latest Release Page](https://github.com/zeridon/ca-certificates-edocs/releases/latest)
